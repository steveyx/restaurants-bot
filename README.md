# restaurants-bot

Commands:
-  to train chatbot in terminal
   rasa train
-  open a new terminal and run rasa server
   rasa run
-  open a new terminal and run action server (for debug mode -vv)
   rasa run actions
-  run chatbot with rasa-x
   rasa x
-  to run chatbot in terminal (debug mode)
   rasa shell --debug
